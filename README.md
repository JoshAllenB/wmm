# WMM Program

## Overview
WMM is a full-stack application for managing client subscriptions, donor data, accounting, and print jobs, with robust backup and export features. It is designed for organizations that need to track clients, manage mailing lists, handle accounting, and automate printing and data exports.

## Features
- **Client Management:** Add, edit, and search clients with detailed information and grouping.
- **Subscription Tracking:** Manage subscription dates, renewals, and classes.
- **Donor and Accounting Modules:** Track donor data and handle accounting operations.
- **Print Management:** Queue, manage, and automate print jobs.
- **Data Export:** Export data for reporting or backup.
- **Backup System:** Automated and manual database backups with retention policies.
- **User Authentication:** Secure login, role management, and activity monitoring.
- **Web Interface:** Modern React frontend with dashboards and admin panels.

## Technology Stack
- **Backend:** Node.js, Express, MongoDB, Socket.IO
- **Frontend:** React, Vite, Material UI, Radix UI
- **Other:** Automated backups, print management, and data export utilities

## Usage
- Start the backend server from the `backend` folder.
- Run the frontend React app from the `client` folder.
- Configure environment variables for MongoDB and other services as needed.
